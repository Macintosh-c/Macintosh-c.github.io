@JSONField 注解说明

导入@JSONField 注解：

import com.alibaba.fastjson.annotation.JSONField;

常见用法：

1. 指定字段的名称

2. 使用format制定日期格式

3. 指定字段的顺序

属性中不使用@JSONField注解输出如下:
{"age":20,"gender":"woman","name":"lily"} //按属性字母顺序排序
使用注解后，上面程序输出如下:
{"name_1":"lily","age":20,"gender":"woman"}
